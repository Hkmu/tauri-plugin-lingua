## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-create-detector-for-all-languages`
- `allow-create-detector-for-languages`
- `allow-detect-language`
- `allow-compute-language-confidence`
- `allow-compute-language-confidence-values`
